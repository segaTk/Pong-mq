# Pong-mq

Требования:
Rust 1.75 или новее: установка
ОС: Windows 10/11, Linux (X11/Wayland), macOS 10.15+

Сборка из исходников:
# 1. Клонируйте репозиторий
git clone <URL-репозитория>
cd pong-mq

# 2. Соберите релиз-версию (оптимизированную)
cargo build --release

# 3. Запустите игру
# Windows:
.\target\release\pong-mq.exe
# Linux/macOS:
./target/release/pong-mq

Быстрый запуск (для разработки)
cargo run --release

Запуск без установки Rust (только Windows)
Скачайте готовый архив с Releases
Распакуйте в любую папку
Запустите pong-mq.exe
Игра не требует установки дополнительных библиотек - всё включено в бинарник!

Зависимости:
Проект использует минимальный набор крейтов:
[dependencies]
macroquad = "0.4"  # Графический фреймворк для 2D-игр

Все зависимости автоматически загружаются и собираются при первой компиляции через cargo.
Структура проекта:
pong-mq/
├── Cargo.toml          # Манифест проекта и зависимости
├── src/
│   └── main.rs         # Исходный код игры (весь в одном файле)
├── assets/             # (опционально) Шрифты, звуки, иконки
│   └── font.ttf        # Кастомный шрифт (если используется)
├── target/             # Автоматически создаётся: бинарники и кэш
├── LICENSE             # Лицензия MIT
└── README.md           # Этот файл

P.S. Игра намеренно реализована в одном файле main.rs для простоты изучения и модификации.
